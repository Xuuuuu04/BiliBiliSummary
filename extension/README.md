# BiliBili AI 助手 - 浏览器插件

这是 **BiliBiliSummarize** 的浏览器扩展版本，允许你直接在 B 站视频页面进行智能总结和文本提取。

## 主要功能
- **智能总结**：基于 AI 的流式视频内容总结。
- **提取文本**：快速获取视频的所有字幕内容。
- **侧边栏支持**：支持 Chrome 侧边栏模式，边看视频边看总结。

## 安装步骤
1. 打开 Chrome 或 Edge 浏览器，进入 `扩展程序` 页面 (chrome://extensions/)。
2. 开启右上角的 `开发者模式`。
3. 点击 `加载解压的扩展程序`。
4. 选择当前项目中的 `extension` 文件夹。

## 使用说明
1. **必须先启动后端服务**：在项目根目录下运行 `python app.py`。
2. 进入任意 B 站视频播放页面。
3. 点击浏览器工具栏的插件图标，或在插件图标上右键选择“打开侧边栏”。
4. 点击“智能总结”或“提取文本”即可。

## 注意事项
- 插件依赖本地运行的 Python 后端（默认地址：http://localhost:5000）。
- 确保你的 `.env` 文件已配置正确的 API Key。
