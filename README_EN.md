# BiliInsight

Bilibili video analysis assistant for subtitle/comment/frame extraction and report generation.

## Language
- Chinese: [README](./README.md)
- English: [README_EN](./README_EN.md)

## Project Structure

## Quick Start
python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && uvicorn asgi:app --reload --host 0.0.0.0 --port 5001
